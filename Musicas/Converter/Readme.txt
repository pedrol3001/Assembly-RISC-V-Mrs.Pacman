---------------------
 CREATING .KARN FILES
----------------------

- Run "MIDI to Text V17 (By Dominic Bodman).exe" on chosen MIDI files.
- Store the resulting text files in the 'MidiText' folder.
- Run "MidiExtractor.exe" to create .karn files in the 'BinaryOutput' folder.
